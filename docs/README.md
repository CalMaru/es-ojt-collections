## Requirements
- npm version: `18.0.0`


## Getting Started - local
1. Set your npm version to `18.0.0`
2. run server

   ```bash
   npm run start
   ```

   or

   ```bash
   make start
   ```

## Getting Started using Docker Container

```bash
make docs
```
