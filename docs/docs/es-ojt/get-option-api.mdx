# Get Option API

import APITable from '@site/src/components/APITable';

뉴스 검색 옵션에 노출되는 `뉴스 분류`, `언론사` 정보를 반환하는 API입니다.

## 기본 정보 {#request-code-info}

| 메서드 | URL |
| --- | --- |
| GET | /search/option |

## 응답 {#response}

| Field | Type | Description |
| --- | --- | --- |
| categories | list (`Category`) | 뉴스 분류 리스트 |
| categories.key | String | 분류 기준 |
| categories.values | list (String) | 분류 리스트 |
| providers | list (`Provider`) | 언론사 리스트 |
| providers.key | String | 언론사 분류 기준 |
| providers.values | list (String) | 언론사 리스트 |
