## 요청 {#request}

### 쿼리 파라미터 {#query-parameters}

:::tip

쿼리 파라미터 내부의 유형 관련 필드 `category_major`, `category_minor`, 언론사 관련 필드 `provider_type`, `provider_location`, `provider_name`에 대한 자세한 정보는 [Get Option API](../../client-api/@docusaurus/get-option-api)를 참고하세요.

:::

```mdx-code-block
<APITable>
```

| Field         | Type   | Description                                                                                                                                                                                        | Required |
| ------------- | ------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- |
| query         | String | 검색어                                                                                                                                                                                             | O        |
| reporter      | String | 기자명                                                                                                                                                                                             | X        |
| start_date    | String | 뉴스 검색 기간 - 시작일<br /><br />**포맷**: `YYYY-MM-DD`<br />**기본값**: `2021-01-01`                                                                                                            | X        |
| end_date      | String | 뉴스 검색 기간 - 종료일<br /><br />**포맷**: `YYYY-MM-DD`<br />**기본값**: `2021-09-30`                                                                                                            | X        |
| category_major | String | 뉴스 유형 - 대분류<br /><br />[Get Option API](/docs/client-api/@docusaurus/get-option-api) 참고                                                                                                           | X        |
| category_minor | String | 뉴스 유형 - 중분류<br /><br />[Get Option API](/docs/client-api/@docusaurus/get-option-api) 참고                                 | X        |
| provider_type | String | 언론사 유형<br /><br />[Get Option API](/docs/client-api/@docusaurus/get-option-api) 참고                                                                                    | X        |
| provider_location | String | 언론사 지역<br /><br />[Get Option API](/docs/client-api/@docusaurus/get-option-api) 참고 | X        |
| provider_name | String | 언론사 이름<br /><br />[Get Option API](/docs/client-api/@docusaurus/get-option-api) 참고 | X        |
| sorting | String | 정렬 기준<br /><br />**입력 가능 항목**: score, latest, old (대소문자 구분 없음) | X        |
