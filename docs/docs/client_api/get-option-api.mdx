# Get Option API

import APITable from "@site/src/components/APITable";

뉴스 검색 옵션에 노출되는 `분류`, `언론사` 정보를 반환하는 API입니다.

## 기본 정보 {#request-code-info}

```mdx-code-block
<APITable>
```

| 메서드 | URL              |
| ------ | ---------------- |
| GET    | `/search/option` |

```mdx-code-block
</APITable>
```

## 응답 {#response}

```mdx-code-block
<APITable>
```

| Field                 | Type                | Description                  |
| --------------------- | ------------------- | ---------------------------- |
| categories            | List (`Category`)   | 뉴스 분류 리스트             |
| categories.all        | String              | 분류 기준 - 전체             |
| categories.category   | List (String)       | 분류 기준 - 유형별 분류      |
| categories.detail     | Dict (list(String)) | 분류 기준 - 유형별 상세 분류 |
| categories.alphabetic | Dict (list(String)) | 분류 기준 - 가나다순         |
| providers             | List (`Provider`)   | 언론사 리스트                |
| providers.all         | String              | 언론사 - 전체                |
| providers.category    | List (String)       | 언론사 - 유형별              |
| providers.detail      | Dict (list(String)) | 언론사 - 언론사 분류순       |
| providers.local       | Dict (list(String)) | 언론사 - 지역별 언론사       |
| providers.alphabetic  | Dict (list(String)) | 언론사 - 가나다순            |

```mdx-code-block
</APITable>
```
