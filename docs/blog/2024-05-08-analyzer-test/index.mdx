## 1. NGram, EdgeNgram 차이 {#test-one}

### NGram {#test-one-ngram}
- 음절 단위로 토큰을 생성하기 때문에 재현율은 높으나 정확도가 떨어짐
- 첫 음절을 기준으로 `max_gram`에서 지정한 길이만큼 토큰 생성


### EdgeNgram {#test-one-edge-ngram}
- 지정한 토크나이저의 특성에 따라 NGram이 일어남 (front, back)

### 실험 - 인덱스 설정 {#test-one-index}
```json title="ngram_analyzer_test.json"
{
  "settings": {
    "index": {
      "number_of_shards": 1,
      "number_of_replicas": 0,
      "max_ngram_diff": 50
    },
    "analysis": {
      "analyzer": {
          "ngram_analyzer": {
            "type": "custom",
            "tokenizer": "ngram_tokenizer",
            "filter": [
              "lowercase",
              "trim"
            ]
          },
          "ngram_edge_analyzer": {
              "type": "custom",
            "tokenizer": "edge_ngram_tokenizer",
            "filter": [
              "lowercase",
              "trim",
              "edge_ngram_filter_front"
            ]
          }
      },
      "tokenizer": {
        "ngram_tokenizer": {
          "type": "ngram",
          "min_gram": "1",
          "max_gram": "50",
          "token_chars": [
            "letter",
            "digit",
            "punctuation",
            "symbol"
          ]
        },
        "edge_ngram_tokenizer": {
          "type": "edge_ngram",
          "min_gram": "1",
          "max_gram": "50",
          "token_chars": [
            "letter",
            "digit",
            "punctuation",
            "symbol"
          ]
        }
      },
      "filter": {
        "edge_ngram_filter_front": {
          "type": "edge_ngram",
          "min_gram": "1",
          "max_gram": "50",
          "side": "front"
        }
      }
    }
  }
}
```

## 2. Nori 사용 시 mixed, discard 차이 {#test-two}

토크나이저가 복합 명사를 처리하게 하는 `decompound mode` 옵션에서 `mixed`와 `discard`가 어떻게 다르게 동작하는지 알아본다.

### mixed {#test-two-mixed}

복합명사로 분리하고 원본 데이터는 삭제

### discard {#test-two-discard}

복합명사로 분리하고 원본 데이터는 유지


### 실험 - 인덱스 설정 {#test-two-index}

```json title="decompound_analyzer_test.json"
{
  "settings": {
    "index": {
      "number_of_shards": 1,
      "number_of_replicas": 0,
      "max_ngram_diff": 50
    },
    "analysis": {
      "analyzer": {
        "korean_mixed_analyzer": {
          "type": "custom",
          "tokenizer": "decompound_mixed_tokenizer",
          "filter": [
            "lowercase",
            "trim",
            "nori_readingform",
            "pos_filter_speech"
          ]
        },
        "korean_discard_analyzer": {
          "type": "custom",
          "tokenizer": "decompound_discard_tokenizer",
          "filter": [
            "lowercase",
            "trim",
            "nori_readingform",
            "pos_filter_speech"
          ]
        }
      },
      "tokenizer": {
        "decompound_mixed_tokenizer": {
          "type": "nori_tokenizer",
          "decompound_mode": "mixed",
          "discard_punctuation": "false"
        },
        "decompound_discard_tokenizer": {
          "type": "nori_tokenizer",
          "decompound_mode": "discard",
          "discard_punctuation": "false"
        }
      },
      "filter": {
        "pos_filter_speech": {
          "type": "nori_part_of_speech",
          "stoptags": [
            "E",
            "IC",
            "J",
            "MAG",
```
