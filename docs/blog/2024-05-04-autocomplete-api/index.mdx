## Autocomplete 기능 요구사항

한글 자동완성만 지원한다고 가정.

1. 한글로 입력했을 때

   1. 몰라 일단 따라해
   2. 검색 결과가 없을 경우
   3. 오타 교정 기능 실팽

2. 영어로 입력했을 때
   1. 자소 변환으로 영어 → 한글 변환
   2. `한글로 입력했을 때`와 동일한 방식으로 자동 완성 지원

### `Elasticsearch 실무 가이드`에서 따라할 수 있는 기능

1. 오타 교정

   `샴성전자` → `삼성전자`
   이 기능은 오타가 발생한 키워드를 교정하는 기능이므로 자동 완성 추천에 필요한가 싶음. 만약에 사용한다면 자동완성 추천 검색어가 없을 때 이 기능을 사용해보면 될 것 같음.

2. 자동완성
3. 한글 자모 분석 필터
4. 한글 초성 분석 필터
5. 한영 오타 변환 필터
6. 한영 오타 변환 필터
7. 스펠링체크 필터
